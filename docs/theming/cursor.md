# Cursor

ZSS contains a class for changing the cursor.

## Source files

- [utilities/_cursor.scss](../../src/utilities/_cursor.scss)

## Generated CSS

```sass
.u-pointer { cursor: pointer; }
```

## Examples

```html
<div class="u-pointer">This element is clickable via JavaScript.</p>
```
